# 📱 Inventory Tracker — Installation auf Handy

Die **Inventory Tracker App** ist eine kostenlose Offline-App zur Verwaltung von Einkauf, Bestand und Verkauf. Alles läuft lokal auf deinem Handy — keine Cloud, keine Daten-Uploads, vollständig privat.

---

## 🚀 Installation

### **Android (Google Play)**

1. Öffne diese URL auf deinem **Android-Handy**:
   ```
   https://phatix.github.io/inventory-tracker/
   ```

2. Warte bis die App geladen hat

3. Tippe oben rechts auf das **Menü (⋮)** oder **Drei-Punkte-Icon**

4. Wähle: **"App installieren"** oder **"Zu Startbildschirm hinzufügen"**

5. Bestätige mit **"Installieren"**

→ Die App erscheint jetzt als Icon auf deinem Startbildschirm! 📲

---

### **iPhone (iOS)**

1. Öffne diese URL auf deinem **iPhone** in Safari:
   ```
   https://phatix.github.io/inventory-tracker/
   ```

2. Tippe unten auf das **Share-Icon** (Pfeil nach oben)

3. Scrolle nach unten und wähle: **"Zum Startbildschirm"**

4. Gib der App einen Namen (z.B. "Inventory") und tippe **"Hinzufügen"**

→ Die App ist jetzt auf deinem Startbildschirm! 📱

---

## ✨ Was kannst du tun?

- 📦 **Produkte verwalten** — Artikel hinzufügen, kategorisieren
- ⬇️ **Einkauf tracken** — Lieferant, Menge, Preis erfassen
- ⬆️ **Verkauf erfassen** — Kunde, Menge, Preis
- 📊 **Statistiken ansehen** — Gewinn, Top-Produkte, Trends
- 📈 **Dashboard** — Schnellübersicht über Bestand & Umsatz
- 💾 **Backup machen** — JSON-Export zum Sichern

---

## 🔒 Datenschutz

✅ **Deine Daten bleiben auf deinem Handy**
- Nichts wird hochgeladen
- Keine Cloud, kein Login
- Keine Werbung, kein Tracking
- Du bestimmst, wann & wo die Daten gespeichert werden

---

## 📋 Wichtig: Regelmäßig Backup machen!

**Damit du deine Daten nicht verlierst:**

1. Öffne die App
2. Gehe zu **Settings** (⚙️)
3. Klicke **"JSON-Export"**
4. Die Datei wird heruntergeladen (speichere sie sicher ab!)

**Backup-Häufigkeit:**
- Täglich (wenn viel los) → sicherer
- Mindestens 1x pro Woche
- Bei wichtigen Änderungen sofort

## 🛟 Erste Schritte

1. **App installieren** (siehe oben)
2. **Produkte hinzufügen** → Dashboard
3. **Einkauf testen** → Ein Produkt einkaufen

---

## 📞 Fragen?

Falls die App nicht funktioniert oder Fragen aufkommen:
- Browser-Cache löschen (Einstellungen → App-Daten → Cache löschen)
- App neu starten
- Auf neuer Version prüfen (wird automatisch aktualisiert)
